四、仓库等配置

  • github上创建一个仓库名：用户名.github.io

  • 配置ssh key，因为github现在不用https传输了，所以使用ssh

  • git config --global user.name "用户名"
git config --global user.email "邮箱地址"
ssh-keygen -t rsa -C '邮箱地址'

  • 将查看到的ssh key添加到github上，头像->settings->SSH and GPG keys

  • 此时需要将本地和github连起来，可以查看官方教程

    • 此处配置项目根目录_config.yml末尾。注意：此处使用的是ssh，以后就可以不用输入密钥了

  • 配置github的Token：settings->Developer Settings->Personal access tokens->Tokens(classic)生成token

  • 安装部署插件hexo-deployer-git npm install hexo-deployer-git --save

  • hexo g	--生成
hexo d	--上传

六、更换电脑后，如何迁移 hexo 博客？

  • 需要的步骤：迁移 hexo 文件夹、git 客户端的下载与安装 、本地Git与Github公钥连接、npm的下载与安装、hexo的下载与安装。

  • 迁移本地 hexo项目 文件夹

    • 需要拷贝原电脑项目文件中的_config.yml，themes，source，scaffolds，package.json，.gitignore 等文件到新电脑新建的文件夹中或者建议直接将原文件夹整个打包拷贝到新电脑上。（注意：也可以从github上copy）

  • 安装 Git 客户端

  • 本地Git与Github公钥连接

    • 生成密钥

    • git config --global user.name "github用户名"
git config --global user.email "github邮箱"
ssh-keygen -t rsa -C "github邮箱"

  • 测试一下吧
